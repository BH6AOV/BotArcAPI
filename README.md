An Arcaea API for bots running in NodeJS. <br/>

## 🤔 Dependences
- node >= `12.16.0`
- npm >= `6.14.0`
- btoa >= `1.2.1`
- node-fetch >= `2.6.0`
- sqlite-async >= `1.0.12`

## 🎉 Run directly
```bash
# optional
$ npm i -g node-pre-gyp

# initialize all node modules
$ npm i

# start service
$ npm start
```

## 🐋 Docker
```bash
# construct docker container from
# compose file and start
$ docker-compose up -d
```
